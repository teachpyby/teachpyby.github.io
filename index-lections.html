    1. Первая цифра после точки

    4. Количество нулей в записи числа
      Программа принимает на вход натуральное число. Необходимо вывести в консоль количество нулей в записи этого числа. Например: a = 903500290 результат - 4
    5. Счастливый билет
      Троллейбусный (трамвайный) билет имеет номер, состоящий из шести цифр. Билет считается счастливым, если сумма первых трёх цифр равна сумме последних трёх, например, 024321
      В единственной строке подаваемой на ввод записано одно целое число. Нужно вывести «YES», если билет с номером N счастливый и «NO» в противном случае.
    6. Угадай число
      Пользователь загадывает целое число N (1 ≤ N ≤ 109), которое ваша программа должна будет отгадать не более чем за 100 попыток. Вы можете делать запросы путем вывода числа из возможного диапазона целых чисел. В ответ на каждый запрос пользователь будет сообщать результат сравнения загаданного числа с числом в запросе.
      Например: Пользователь загадал 20 Вариант вывода программы: 25 Пользователь вводит: < 12 Пользователь вводит: > ... 20 Пользователь вводит: = Ура, я отгадал!
  • 003 Cписки
    Задачи
    1. Проверить, является ли слово, строка, число палиндромом
      Палиндром — это строка(или число), которое можно прочитать одинаково справа налево, либо слева направо. К примеру, слово «кот» не является палиндромом, а слово «потоп» является палиндромом. Также и с числами: число 12314 — не палиндром, число 345543 — палиндром. На вход поступает строка. Вывод - ответ YES/NO.
    2. Склеить две упорядоченные последовательности в одну.
      На вход поступает 2 упорядоченные последовательности Например [1, 2, 3, 10, 20] [2, 4, 5, 11, 20, 30] Вывод: [1, 2, 2, 3, 4, 5, 10, 11, 20, 20, 30]
